# Wings of Fire Chatbot

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ai-chatbots/dragonchat.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dragonchat
   ```
3. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up your API keys in the `.env` file.

## Usage Guide
1. Start the chatbot by running:
   ```bash
   python main.py
   ```
2. Follow the prompts in the terminal to interact with the Wings of Fire chatbot.
3. For more advanced options, refer to the documentation in the `docs` folder.
